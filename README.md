# Compendium
An Educational Tool For Exploring Plants. This application utilizes image search technology, utilizing photos inputed from the user to discover plants, including common or scientific names.
Sign in, and create a personal collection to easily reference searched results and make and save comments.

Learn the value of discovering plants on hikes, foraging opportunities, and enhancing gardening knowledge.
Get your own API key from [here.](https://documenter.getpostman.com/view/24599534/2s93z5A4v2)
